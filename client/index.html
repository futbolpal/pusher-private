Pusher-Private

Pusher.com is a websocket cloud based service to provide an easy way to send push notifications from your web app to a group of clients.

However its libraries do not have a built in way to easily send a message to the current client making a request to your server or app

socket.io contains a unique client id when each client connects to the server. On connection, the Pusher channel information (one private and one public) is sent to the client. The client can now send a socket.io message 'trigger-public' or 'trigger-private' to request a public or private Pusher trigger respectively.
